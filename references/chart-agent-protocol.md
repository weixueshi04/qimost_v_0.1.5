# Chart Agent Protocol

Use this for engineering, quantitative, chart-heavy, table-heavy, or diagram-heavy courses.

## Required Artifacts

Create a figure inventory when charts matter:

```markdown
| Figure ID | Source | Type | What it shows | Related K IDs | Exam use | Extraction quality |
|---|---|---|---|---|---|---|
```

## Question Design

For chart-based questions:

- identify axes, units, variables, and assumptions,
- cite the source figure,
- ask about interpretation, calculation, comparison, or error traps,
- include a solution path that teaches how to read the figure.

Do not invent numeric values that are not visible or derivable.
