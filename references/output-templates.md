# Output Templates Reference

Use this when creating course artifacts.

## Major Artifact Header

```markdown
Current phase:
Purpose:
Inputs used:
Source coverage gate result:
Source priority used:
Original-source verification:
Quality Check:
Correction/evolution status:
Follow-up task status:
Knowledge IDs covered:
Source evidence:
Next action:
State update:
Learnability check:
```

## Compact Answer Template

```markdown
Source verification:
Relationship map:
Plain-language explanation:
Course-first reasoning:
Special case:
Exam-ready answer:
Quick recall:
```

Use only the fields that fit the artifact; do not add empty filler.
