# Full Workflow Reference

Use this when running the complete 10-12 hour Qimo Speedrun path.

## Phase Order

1. Capability check.
2. Workspace initialization.
3. Source coverage.
4. Current exam rules lock.
5. Evidence index and source priority map.
6. Knowledge framework.
7. Courseware question bank.
8. Homework mapping.
9. Exam 1 and review.
10. Exam 2 and review.
11. Recitation and sleep protection.
12. Exam 3 and final sprint.
13. Post-run retrospective.

Do not skip a phase unless the user explicitly accepts the risk.

## Study Block Pattern

Each block should end with:

- current phase,
- artifact produced,
- weak points discovered,
- next allowed action,
- whether a follow-up task should be proposed.

## Emergency Compression

If time is short:

- 2-4h: source coverage, exam rules, must-know framework, 20-30 high-yield questions.
- 6-8h: framework, course questions, Exam 1, mistake review, recitation.
- 10-12h+: full three-exam path plus sleep and morning retrieval.
