# Auto Intake Reference

Use this when the user provides one unsorted folder or knowledge-base dump.

## Classification

Classify every item into:

- current exam rules,
- review recordings/transcripts,
- PPT/courseware,
- textbook/reference,
- homework/in-class exercises,
- past exams,
- senior notes,
- figures/tables/charts,
- other.

Assign S0-S6 priority immediately. Do not wait until synthesis.

## Ledger Requirement

Write each item to `logs/source_coverage.md` with:

- file/item name,
- type,
- priority tier,
- extraction method,
- read status,
- evidence locations,
- action needed.

Unread S0/S1/S2 items block synthesis unless the user explicitly accepts lower confidence.
